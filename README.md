# Auction Customer App

Customer-facing Next.js web application for the Vulnerable Auction Lab platform.

## Overview

This application allows customers to browse auctions, place bids, manage orders, and track shipments.

## Quick Start

```bash
npm install
npm run dev
```

The application will be available at `http://localhost:3000`
